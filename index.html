GithubPages+Hexo搭建博客

目标效果

需要的环境和介质

  1. node.js
  2. hexo

    创建githubpages项目

  3. 注册github
  4. 创建githubpages项目，名字为.github.io
  5. 项目中添加html页面后，就可以用浏览器打开.github.io

配置sshkey

  • 进入gitbash下
  • 查看是否存在.ssh文件ls -al ~/.ssh
  • 如果存在ssh文件就删除rm -rf ~/.ssh
  • 生成ssh文件 ssh-keygen -t rsa -C “angelen10@163.com”（填入自己github注册邮箱），需要输入密码就enter跳过
  • eval ssh-agent -s
  • ssh-add
  • 拷贝sshkey clip < ~/.ssh/id_rsa.pub
  • 然后到github下的setting下面去添加ssh key

安装hexo

  1. windwos环境下进入git bash
  2. npm install -g hexo （-g指全局）
  3. 创建hexo项目文件夹
  4. 进入文件夹，输入hexo init
  5. 安装依赖包，npm install
  6. hexo generate
  7. hexo server
  8. 浏览器输入localhost:4000 查看

    hexo与githubpages关联

    更改配置文件


    主义hexo3.0以后deploy 的type要填git
    hexo generate
    hexo deploy
